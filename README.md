<h3>Getting Started</h3>

<h4>Sever</h4>

Github 저장소를 clone하고 해당 디렉토리에서 터미널을 열고 아래 명령어를 차례로 입력합니다.

```
$ cd server
$ npm i
$ npm start
```

서버는 [http://localhost:3001](http://localhost:3001)에서 실행됩니다.
